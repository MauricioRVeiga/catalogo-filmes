# Catálogo de Filmes

Projeto full stack com Flask, Flet e landing page estática.

## Como rodar

### Backend
```bash
cd backend
python app.py
```

### Frontend
```bash
cd frontend
flet run main.py
```

### Landing page
Abra o arquivo `landing/index.html` no navegador.
