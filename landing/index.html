Catálogo de Filmes

Aplicação full stack com Flask, Flet e landing page estática. O projeto exibe filmes, cadastra novos registros e documenta a API com Swagger.

Como executar

  1. Instale as dependências do backend e frontend.
  2. Rode o Flask para expor a API.
  3. Rode o Flet para abrir a interface.
  4. Acesse o Swagger em /api/docs.

Recursos

  • Backend com Blueprints.
  • 2 endpoints GET documentados.
  • 1 endpoint POST com Pydantic.
  • Frontend consumindo a API.
