Catálogo de
Filmes

Aplicação full stack para gerenciar um acervo de filmes favoritos. Backend REST com Flask, interface desktop com Flet e documentação automática via Swagger UI.

Flask + Flask-RESTX Flet Pydantic Swagger UI Tailwind CSS

Como executar

Siga os passos abaixo para rodar o projeto localmente.

1

Instalar dependências

Crie um ambiente virtual e instale os pacotes.

python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux / Mac
source .venv/bin/activate

pip install -r requirements.txt
2

Iniciar o Backend

Suba o servidor Flask na porta 5000.

cd backend
python app.py

# API em:
# http://127.0.0.1:5000/api

# Swagger em:
# http://127.0.0.1:5000/api/docs
3

Iniciar o Frontend

Abra a interface desktop com Flet em outro terminal.

cd frontend
flet run main.py

# A janela desktop abre
# automaticamente conectada
# ao backend Flask
📄

Landing Page

Abra o arquivo landing/index.html diretamente no navegador — não requer servidor.

Endpoints da API

Documentação interativa disponível em /api/docs (Swagger UI).

GET

/api/movies

Lista todos os filmes cadastrados no catálogo. Retorna 200 com array de filmes.

GET

/api/movies/{id}

Retorna um filme pelo ID. Retorna 200 se encontrado ou 404 caso contrário.

POST

/api/movies

Cadastra um novo filme. Valida os dados com Pydantic antes de persistir. Retorna 201 em caso de sucesso ou 400 com detalhes dos erros.

title · string 1–100 genre · string 1–60 year · int 1888–2100 rating · float 0–10
